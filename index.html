The Main Effect
Set a Reference Number above (e.g. 4050). The spectator enters their number (e.g. 4058). The app subtracts the Reference from the Input (4058 - 4050 = 8). The result "8" is revealed as the "Phone Password".
Notes Mode
Instead of a number, the phone unlocks to a Note. This reveals the Spectator's Name and Zodiac (if errors triggered).
Ghost Lock
Tap anywhere on the background to create fake dots. Tap a Key to register a real dot. Unlocks on the final digit.
